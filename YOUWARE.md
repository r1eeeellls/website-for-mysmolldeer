# Birthday Website Project

## Project Overview
A multi-page interactive birthday website with animations, music, and falling symbols effect.

## File Structure
- `index_mobile_fixed.html` - Main HTML file with embedded CSS and JavaScript
- `1.gif` - First page animated icon
- `write.gif` - Second page envelope animation
- `Party.mp3` - Background music
- Audio files for click sounds are hosted externally

## Key Features
- Three-page navigation system with smooth transitions
- Typewriter effect for letter content
- Falling heart and star animations
- Background music with toggle control
- Confetti effects
- Click sound feedback
- Mobile-responsive design

## Color Scheme
- Background: Light purple (`#f0f0ff`)
- Primary text: Slate blue (`#6a5acd`)
- Highlights: Blue violet (`#8a2be2`)
- Buttons: Medium orchid (`#9370db`) with hover effect
- Decorative elements: Purple tones

## Technical Implementation
- Pure HTML/CSS/JavaScript (no build process required)
- Canvas Confetti library via CDN
- CSS animations for falling effects
- Audio elements for background music and interactions
- Page transitions using opacity and z-index

## No Build Required
This is a static website that runs directly in browsers without any compilation or build steps.
